# T&S Career Roadmap Dashboard

An interactive dashboard built with React and Vite mapping my learning path toward a Trust & Safety Engineer role.

## What it does
- Visual roadmap across 5 phases of T&S engineering skills
- Tracks tools and their current status
- Links to every resource needed for each phase

## Built with
- React
- Vite
- JavaScript

## How to run
npm install
npm run dev
